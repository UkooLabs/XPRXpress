What Is XPR Express?

XPR Express is a tool for extracting XBMC XPR files (non-protected) back into their original image files or for compressing image files into the XBMC XPR file format.

Additional Features

Shell integration
Image Preview
Instant Preview

Extract Usage

Simply select the Extract tab then select an appropriate source XPR file. Once done you may need to click Load XPR depending on how you selected the XPR. Once your XPR is loaded you will then be presented with a tree view of the XPR file contents. You then can either right click the appropriate file in the tree to preview it or if you select a destination folder you will have the option to either extract the selected file or all files.

 Compress Usage
Simply select the Compress tab then select the quality setting you require. If you wish the generated XPR to be not protected so that people can freely use your graphics then tick the No Protect box. Tick Only Swizzled if you require faster rendering of graphics in XBMC but dont mind about larger memory use. You can then select your source folder containing your images, then the destination XPR file name for them to compress to.

 Command Line

Commandline usage of XPR express is very similar to that of XBMCTex.

-help or -h

Shows this screen.

-input <dir/xpr> or -i <dir/xpr>
Input directory or XPR file.

-output <dir/xpr> or -o <dir/xpr

Output directory or XPR file. Default is current directory and Textures.xpr if input is a folder, other wise current directory.

-reqoutputpath or -rp

Causes XPR Express to request an output path for extraction.

-reqoutputfile or -rf

Causes XPR Express to request an output file for compression.

Note: If you supply a xpr file for input then a folder is expected for output and vice versa.
